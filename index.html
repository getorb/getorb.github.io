Questions

Frequently asked questions.

Do I need to pay for an AI subscription?
No. Orb works on-device for free, and you can add free API providers or run a local model on your own hardware. A paid AI account is one option among several, never a requirement.
Is my data private?
On-device data stays on your device. If you self-host, the server is your machine and you only ever connect to your own. There is no developer cloud — nothing of ours could read your data, because nothing of ours exists.
Does it work offline?
Yes — the on-device brain runs with no internet. Features that need the cloud or your PC won't be available offline, and Orb tells you when that's the case instead of failing silently.
What does the server actually add?
Hands and memory: real work on your own computer, proactive briefings, hands-free voice, reminders that survive restarts, and agents you can watch from your phone. Run it in ~15 minutes.
Will it sound different depending on which AI model runs it?
No. You set Orb's personality once and it holds across every brain — the model is background infrastructure; the personality is constant.
Is the code open source?
The entire server is — MIT-licensed at github.com/getorb/Orb-Backend: the voice loop, the agent, the proactive engine, the mind. The iOS app is closed-source; the API is documented, so you could even build your own client.
